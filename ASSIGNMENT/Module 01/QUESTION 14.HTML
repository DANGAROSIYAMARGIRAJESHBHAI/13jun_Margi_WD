QUESTION 14:-

What is the use of a span tag? Explain with example?
 

ANSWER:-

  
    → Span Tag :

    - The span tag is used for the grouping of inline elements 

    - This tag does not make any visual change by itself. 

    - Span is very similar to the div tag, but div is a block-level tag and span is an inline tag.

    - span elements do not have a value property

    - A span tag is written with an opening and a closing tag, like so:
      <span>Content goes here</span>

          

   Example :

   -  <p>We can use span tags to target text with CSS styling, like with <span id="red-text">
      this red text</span> <p>

   Result:

    - We can use span tags to target text with CSS styling, like with  red text
