QUESTION 14:-

What is the use of a span tag? Explain with example?
 

ANSWER:-

  → Span Tag :

- the span tag is a generic inline container element.
- You use this element to wrap sections of text for styling purposes or to add attributes to a section of text without creating a new line of content.
- we use span tag for inline element and div for block element.

Example :
- <span id="red-text">
this red text</span>

Result:

- this red text
