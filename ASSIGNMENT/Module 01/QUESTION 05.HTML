QUESTION 05:-

What are different types of lists in HTML? With Example.
 

ANSWER:-

  → Mainly three types of list in HTML.

- 1) Ordered list
2) Unordered list
3) Data description list

1)Ordered list:

- If need show order in the list that time use order list tag in the HTML.
- It uses numbers or some short of notation that indicates a series of items.
for example : an ordered list can start with number 1, and continue through 2,3,4, and so on.

→ <ol></ol> : Ordered list
<li></li> : list items

Example:-

ORDERED LIST(MONTH NAME)

1.JANUARY
2.FEBUARY
3.MARCH
4.APRIAL
5.MAY

2)Unordered list:

- It is used to create a list of items in no particular order.
- the order of items is not relevant.
- by default the items in the list will be marked with bullets.

→ <ul></ul> : Unordered list
<li></li> : list items
Example:-

UNDERED LIST(BIRDS NAME)

■ Crow
■ Duck
■ Peacock
■ Eagle
■ Sparrow

3)Data description list:

- It is a list of term with a description of each term.
→ <dl></dl> : Data list
<dt></dt> : Data term
<dd></dd> : Data description

Example:-

1.India
- Largest Democracy in The World.

2. America
- The world's military superpower and one of the most affluent nations on the planet to being home to one of the most diverse populations and the No. 1 global exporter of entertainment, there is no shortage of truly remarkable things to be said about the United States.
3. Australia
- More than 80% of our plants, mammals, reptiles and frogs are unique to Australia and are found nowhere else in the world. Some of our Australian animals are very well known like kangaroos, dingos, wallabies and wombats and of course the koala, platypus and echidna.
