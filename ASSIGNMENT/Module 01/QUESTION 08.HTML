QUESTION 08 :-

What are the various formatting tags in HTML?
 

ANSWER:-

  
 → Formating Tags:-


    • html provides us with the ability for formatting text just like we do in MS word
      or any text editing software.

    • It give us the ability to format the text without using CSS.

    • The following HTML tags are used to format the appearance of the text on your web page.


EXAMPLES :-


    
   1. <b>       - Bold text
      <strong>  - Bold text

   2. <br>      - single line break

   3. <i>       - Italic text 
      <em>      - Italic text(emphsize tag) 
      <cite>    - Italic text(cite tag)

   4. <u>       - under line
      <ins>     - under line 

   5. <del>     - delet
      <strike>  - delet

   6. <big>     - increase text size

   7. <small>   - dicrease text size

   8. <mark>    - highlight 

   9. <sub>     - subscripted text

  10. <sup>     - superscripted text

  11. <hr>      - horizontal raw

  12. <center>  - center typeing

  13. <abbr>    - abbreviation tag

  14. <q>       - quted text
