QUESTION 10:-

 
How can we club two or more rows or columns into a single row or column in an HTML table? With Example.
 

ANSWER:-

  
        - It can be done by using the rowspan and colspan attribute in HTML. 

                    → Rowspan:

                     - The rowspan is used to merge or combine the number of cells in a row.
                     
                      Example:-  <td rowspan="2">24</td>

                            


                    → Colspan:

                      - colspan attribute is used to merge two or more columns to form a single column.
                                
                       Example:- <td colspan="4">sum :$18</td>
