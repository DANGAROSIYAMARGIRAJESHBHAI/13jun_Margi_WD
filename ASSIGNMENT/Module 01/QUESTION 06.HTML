QUESTION 06 :-

What is the ‘class’ attribute in HTML? With Example.
 

ANSWER:-

  
 → "Class" :-
         
        - A same name of Class can be used by multiple HTML elements.

        - Class name is sensitive.

        - Use multiple class on the same element.

        - We define class name what we are desire.

         

        Example :<h2 class="title"> hello HTML </h2>
                 <title class="title"> hello HTML </title>

                 - in this example title is the name of class and 
                   class is a attribute.
