QUESTION 19:-

 
What is difference between HTML and XHTML?
 

ANSWER:-

  
            → HTML:

            - HTML stands for Hypertext Markup Language.

            - Tags are not case sensitive.

            - Tags are not extensible.

            - HTML is less expressive

            - There is no hard rule on the structure of the elements.

            - attribute values are not significant in html. 



            → XHTML:

            - HTML stands for Extensible Hypertext Markup Language.

            - Only lower case tags are allowed.

            - Tags are extensible.

            - XHTML is more expressive compared to HTML.

            - The structure of the elements should be followed.

            - attributes value are important in XHTML.
