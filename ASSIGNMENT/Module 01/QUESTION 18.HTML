QUESTION 18:-

 
What is SVG?
 

ANSWER:-

  
      →  SVG:-


            - SVG stands for Scalable Vector Graphics

            - HTML SVG is used to describe the two dimensional vector  graphics.
            
            - HTML SVG is a modularized language which is used to describe graphics in XML.
            
            - It describe two-dimensional vector and mixed  graphics in XML.
            
            - SVG is mostly used for vector type diagrams like pie charts, 2-Dimensional graphs in an X,Y coordinate system etc.
            
            - The <svg> element specifies the root of a SVG fragment.
