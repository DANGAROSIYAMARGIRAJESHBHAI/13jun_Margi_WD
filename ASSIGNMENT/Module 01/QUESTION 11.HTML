QUESTION 11:-

What is the difference between a block-level element and an inline element?
 

ANSWER:-

  - A block-level element always starts on a new line and takes up the full width available. An inline element does not start on a new line and it only takes up as much width as necessary.

→ Block level element :

- begins a new line of text.
- you can set the width and height values.
- Example : <p> ,<h1> ,<div> ,<pre>


→ Inline element :

- does not begin a new line of text.
- you can't set the width and height values.
- Example : <span> ,<a> ,<strong> ,<img>
