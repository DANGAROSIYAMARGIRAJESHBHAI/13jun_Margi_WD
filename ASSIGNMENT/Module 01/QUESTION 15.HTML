QUESTION 15:-

 
How to insert a picture into a background image of a web page? With Example.
 

ANSWER:-

  

           -  Background images in HTML can be added using the background-image property in CSS, internal style sheet,
              or the background attribute in HTML .

           -  we use two way to insert background image. 

               1)absolute path   
               2)relative path

           - in this attribute we also set height or width of background image.   


           - Example :-

            <body>
                <div style="background-image: url('https://www.website.com/images/image_background.jpg');> 
            </body>
