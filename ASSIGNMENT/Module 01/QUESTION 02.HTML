QUESTION 02 :-

What are tags and attributes in HTML?
 

ANSWER:-

  
→  Tags:  

 - A tag is a short description of an HTML object and links.
 - tags have two types.
    1)pair tag   
    2)non pair tag

 - the first tag in pair is start tag and the secound tag is end tag.

 - Example : <html></html>
             <head></head>
             <body></body>

→  Attributes: 
   
 - attributes is a additional information of that tag.

 - attributes always pass in the start tag.

 - html attributes can be used to change the color , size or functionality of html elements.

 - Example : <p align="left">....</p> 
             - In this example align is the Attribute name and left is the value.
