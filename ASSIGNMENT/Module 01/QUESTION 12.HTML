QUESTION 12 :-

How to create a Hyperlink in HTML? With Example.
 

ANSWER:-

  → Hyper link:-

- The <a> tag defines a hyperlink.
- It is used to link from one page to another.
- The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
- By default, links will appear as follows in all browsers: An unvisited link is underlined and blue.

1. Text hyperlink : Uses a word or phrase to take visitors to another page, file or document.
2. Image hyperlink : Uses an image to take visitors to another page, file or document.

Example : < href = "www.google.com"> google </a>
