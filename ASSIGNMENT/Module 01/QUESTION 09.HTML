QUESTION 09 :-

How is Cell Padding different from Cell Spacing? With Example.
 

ANSWER :-

  - The main difference between cellpadding and cellspacing is that cellpadding refers to the space between the cell boundary and the printed text.
While cell spacing refers to the distance between each individual cell in the table.

- Both of these characteristics are utilised in table formatting.

→ Cell Padding:-

- Cell padding is the space between the cell edges and the cell content.
- use the inline style attribute, to set cell padding in HTML.
- Example : <table cellpadding="value" >.....</table>

→ Cellspacing:-

- The cellspacing attribute is used to set the distance or the space between the edges of the cells of the table.
- Cell padding is an inline attribute used under the table tag.
- Example : <table cellspacing="value" >.....</table>
