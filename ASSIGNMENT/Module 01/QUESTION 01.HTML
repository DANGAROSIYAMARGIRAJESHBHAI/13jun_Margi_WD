QUESTION 01 :-

Are the HTML tags and elements the same thing?
 

ANSWER :-

  → 

Both are different ELEMENT includes starting tag , content and ending tag... when TAGS not includes contents.



1. Tags:-

- A tag is a short description of an HTML object and links.
- Tags have two types :

1)Pair tag - must be closed
- Example :<html></html>,<head></head>
2)Non pair tag - Don't need to closed.
- Example :<br>,<hr>,<meta>


2. Elements:-

- Combination of starting tag, ending tag and contents.
- Example : <title>hello html</title>

- Elements have to types :

1)Inline element - Inline elements are displayed on the same line. they do not start on a new line.
- Take up only as much width as their contents require.
- Example :<<b>,<strong>
2)Block level element - Block element take up the whole horizontal space available in its container.
- They start on a new line and take up as much height as their contents require.
- Example : <h1>,<div>
