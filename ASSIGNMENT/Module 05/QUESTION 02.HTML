Question 02 :-

How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

jQuery has several methods for CSS manipulation

1) addClass()

- Adds one or more classes to the selected elements

example : $("button").click(function(){
$("#div1").addClass("important blue");
});


2)removeClass()

- Removes one or more classes from the selected elements

example : $("button").click(function(){
$("h1, h2, p").removeClass("blue");
});


3)toggleClass()

- Toggles between adding/removing classes from the selected elements

example : $("button").click(function(){
$("h1, h2, p").toggleClass("blue");
});


4)css()

The css() method sets or returns one or more style properties for the selected elements.

example :$("p").css("background-color");
