QUESTION 17 :-

What is the meaning of the CSS selector?
 

ANSWER :-

 

→   In CSS, selectors are used to target the HTML elements on our web pages that we want to style.

→   CSS have many types of selectors.

→ Simple Selector:

It is used to select the HTML elements based on their element name, id, attributes, etc. Element Selector. Id Selector. Class Selector.

- Class selectors
- Element selectors
- Id selector



→ Combinator selectors:

Combinator selector select element based on their relationship between them.

- Group selectors
- Child selectors
- General selectors
- Adjacent selectors



→ Pseudo-class selector :

Pseudo-class selector apply eith colon(:) symbol.

- Link selector
- Hover selector
- Focus selector



→ Pseudo-class selector :

Pseudo-element selector apply eith colon(::) symbol.

- Placeholder selector
- After-Before selector
