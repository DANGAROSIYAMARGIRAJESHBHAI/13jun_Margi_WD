QUESTION 16 :-

What are the advantages and disadvantages of using external style sheets?
 

ANSWER :-

 

→   Disadvantages:-

→   The execution of the file may be deferred till the external style sheet is loaded.

→   While implementing style sheets, we need to test Web pages with multiple browsers in order to check compatibility issues.



→   Advantages:-

→   There is no restriction to the number of web pages that can use the external style sheet.

→   They can also assist in giving each page to which they have linked a consistent format.

→   The look can be instantly applied to every single web page using an external style sheet.
