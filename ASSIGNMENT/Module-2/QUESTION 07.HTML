QUESTION 07 :-

How can image repetition of the backup be controlled?
 

ANSWER :-

 

→   We can use background repeat proparty for image repetition .

→   repeat-x: This property is used to repeat the background image horizontally.

→   repeat-y: This property is used to set the background image repeated only vertically.

→   We can use background no-repeat proparty for not image repetition. displays the background image only once.
