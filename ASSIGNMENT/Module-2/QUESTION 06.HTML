QUESTION 06 :-

How can the background color of an element be changed?
 

ANSWER :-

 
→   We can use background color property for set the background color.

→   Background-color values can be expressed in many ways.

→   Like:
     1) hexadecimal values __ Ex. #FFFFFF, #000000
    2) rgb __ Ex. (255,0,0)
    3) named colors __Ex. white, black etc.
