QUESTION 14 :-

What is embedded style sheets?
 

ANSWER :-

 

→   Inline styles are those that are used as part of the HTML tag itself. Embedded styles are located in the header of a page, and apply to that entire page .

→   embedded style sheets have some benefits also.

→   they load immediately with the page itself

→   This technique can be a benefit from a download speed and performance perspective.


Syntax : 
        <head>
        <link rel="stylesheet" href="./css/file1">
        </head>
