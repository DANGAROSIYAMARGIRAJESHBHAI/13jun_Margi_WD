QUESTION 05 :-

What do you understand by CSS opacity?
 

ANSWER :-

 
→   The opacity in CSS is the property of an element that describes the transparency of the element.

→   In the image opacity property is used for transparency of the image.

→   The value of opacity in between 0.0 to 1.0.

→   In the value of opacity 0.0 is fully transparent and 1.0 is fully opaque.
