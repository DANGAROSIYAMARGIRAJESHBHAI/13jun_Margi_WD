QUESTION 12 :-

How to maintain the CSS specifications?
 

ANSWER :-

 

→   The Specification defines how CSS properties should be implemented by browser vendors along with detailed algorithms, code samples and tabular information.

→   The Specification also include The syntax and data types of the language.

→   It is a detailed explanation on CSS.

→   The CSS specifications are maintained by the World Wide Web Consortium (W3C).

→   Even though every browser supports CSS, there are many inconsistencies in the supported specification version. Some browsers even have their own implementation of the specification and have proprietary (vendor) prefixes.
