QUESTION 19 :-

What is the rule set?
 

ANSWER :-

 
- Rule set include selector and declaration block .

- The selectors points to the HTML element you want to style

- Each declaration includes a CSS property name and a value separated by colon.

- Multiple CSS declarations are separated with semicolons and declaration blocks are surrounded by curly braces.

        Example:-
        div{
            color: black;
            background color: white;
          }
        HTMl elements you want to style
        color and background color = property.
        black and white = property value
