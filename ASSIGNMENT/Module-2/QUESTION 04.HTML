QUESTION 04 :-

Name a few CSS style components.
 

ANSWER :-

 
→ There are mainly three types of components in css style.
   1.Properties
   2.Value
   3.Selector

1.properties :-

Example - color, border, margin, font-style, font family, background color , ect...

2.value :-

→ we use some specific character for define perticular properties each character khow as a value.
Example - color:red (red is a value),
                  border:1px solid (1px solid is a value)

3.selector:-

→  There are several different types of selectors in CSS.
css element selector.(element name{__})
css id selector.(#id name{__})
css class selector.?(.calssname{__})
css universal selector.(*{__})
css group selector.(element name , element name {__})
