QUESTION 01

What are the advantages of Bootstrap?

ANSWER :-

Bootstrap is Easy to Use

Bootstrap is very easy and quite simple to use for designing and development.

Fast and Time-Saving Framework:

Bootstrap is an agile framework which is quite faster than other frameworks. It saves time due to its standard ready-made coding blocks

A Pack of JavaScript Fundamentals:

Bootstrap supports Javascript Components so one don't require knowledge of scripting. It comprises JavaScript components

Responsive features

Bootstrap's responsive CSS adjusts to phones, tablets, and desktops.
