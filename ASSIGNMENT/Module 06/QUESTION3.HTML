QUESTION 03

What are the default Bootstrap text settings?

ANSWER :-

Bootstrap uses a default font-size of 16px

Its line-height is 1.5.

The default font-family is "Helvetica Neue", Helvetica, Arial, sans-serif.

In addition, all <p> elements have margin-top: 0 and margin-bottom: 1rem

<h1> - <h6> is also default font-size in bootstrap

h1 Bootstrap heading - (2.5rem = 40px)

h2 Bootstrap heading - (2rem = 32px)

h3 Bootstrap heading - (1.75rem = 28px)

h4 Bootstrap heading - (1.5rem = 24px)

h5 Bootstrap heading - (1.25rem = 20px)
h6 Bootstrap heading - (1rem = 16px)
