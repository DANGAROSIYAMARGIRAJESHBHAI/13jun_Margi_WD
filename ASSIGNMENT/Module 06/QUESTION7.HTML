QUESTION 07

How can you use Bootstrap to make thumbnails?

ANSWER :-

In Bootstrap, Thumbnail is a border which is around the image. It is created using the .img-thumbnail class.

Syntax:

<img src="..." alt="..." class="img-thumbnail">

Rounded Corners :- The .img-rounded class adds rounded corners to an image

Circle :- The .img-circle class shapes the image to a circle
