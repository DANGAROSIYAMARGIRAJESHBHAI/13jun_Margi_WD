QUESTION 09

How can one create an alert in Bootstrap?

ANSWER :-

Bootstrap Alerts are used to provide an easy way to create predefined alert messages.

Alert adds a style to your messages to make it more appealing to the users.

Alerts are created with the .alert class

Add the alert-link class to any links inside the alert box to create

To close the alert message, add a .alert-dismissible class to the alert container.

The .fade and .in classes adds a fading effect when closing the alert message
