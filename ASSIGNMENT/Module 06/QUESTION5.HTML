QUESTION 05

What is the difference between Bootstrap 4 and Bootstrap 5

ANSWER :-

Bootstrap 4 :-

Bootstrap 4 has less collection of colors.

It has 5 tier grid system.

Bootstrap 4 provides normal form layout with no advance features.

bootstrap 4 provided Small icon collection.

Bootstrap 5 :-

bootstrap 5 Has vibrant and more colors available.

It has 6 tier grid system.

It Provides advance and attractive features to make form

bootstrap 5 provided Large icon collection.

In this way bootstrap 5 is upgrated version of bootstrap4.
