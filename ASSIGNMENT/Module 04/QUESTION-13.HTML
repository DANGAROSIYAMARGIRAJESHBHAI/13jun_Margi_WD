Question 13 :-

What is the function of the delete operator?

delete operator used for delete proparty from an object.

On successful deletion, it will return true , else false will be returned

Delete is comparatively a lesser-known operator in JavaScript.

Syntax :

delete object
or
delete object.property
or
delete object['property']

Example :-

let introduction = {
firstName: "margi",
lastName: "patel",
salary: 40000
}

console.log(delete introduction.salary);
console.log(introduction);

Output :-

true
{"firstName":"margi","lastName":"patel"}
