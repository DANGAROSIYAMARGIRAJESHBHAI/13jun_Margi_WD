Question 08 :-

What is === operator?

This operator allows for a much stricter comparison between the given variables or values.

"===" operator is also know as strict operator.

This operator returns true if both variable contains same information and same data types otherwise return false.

It checks not only the values but also types of two variables.

Example :-

10 === 10 : (true)

10 === '10' : (false)
