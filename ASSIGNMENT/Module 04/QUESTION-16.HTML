Question 16 :-

How can a page be forced to load another page in JavaScript?

In JavaScript, we can use window. location object to force a page to load another page. We can use the location object to set the URL of a new page.

It is a reference to a Location object that is it represents the current location of the document

We can change the URL of a window by accessing it.

Syntax :

<script>
window.location = <Path / URL>
</script>
