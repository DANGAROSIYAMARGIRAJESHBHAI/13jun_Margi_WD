Question 11:-

What are all the looping structures in JavaScript?

In programming, loops are used to repeat a block of code.

By default loop's starting index is '0'.

By default loop is execute infinite times.

Loop have mainly two types. 1) entry control loop 2) exit control loop

Entry control loop :-

i.) For loop
ii.) While Loop
iii.) For-in Loop
iv.) For-of Loop
v.) For-each Loop

Exit control loop :-

i.) Do-While loop
