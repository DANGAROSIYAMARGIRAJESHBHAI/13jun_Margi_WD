Question 06 :-

Write the code for adding new elements dynamically?

we can adding new elements dynamically by javascript.

Code :-

< Body >

< div id = "parent" >
< p id ="line1" > first line < /p >
< p id ="line2" > secound line < /p >
< /div >

< script >
var pre=document.createElement('p')
var node=document.createTextNode('third line')
pre.appendChild(node)
var parent=document.getElementById("parent")
parent.appendChild(pre)
< /script >

< /Body >



Output :-

first line

secound line

third line
