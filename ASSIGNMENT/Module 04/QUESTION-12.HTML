Question 12:-

How can you convert the string of any base to an integer in JavaScript?

In JavaScript parseInt() function or method is used to convert the string parameter or value to an integer value .

Example :-

parseInt("10"); = 10

parseInt("10.255"); = 10

parseInt(" 10 "); = 10

parseInt(" number 10 "); = NaN

parseInt("10" , 3); = NaN
