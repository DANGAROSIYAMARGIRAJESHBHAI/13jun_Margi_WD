How can the style/class of an element be changed?

There are two common approaches that changed the style/class of an element.
1)style.property
2)Changing the class itself

Changing CSS with the help of the style property.

Syntax:document.getElementById("id").style.property = new_style

We can use two properties that can be used to manipulate the classes

The classList Property and The className Property.

Syntax:

document.getElementById("id").classList

document.getElementById("id").className = class
