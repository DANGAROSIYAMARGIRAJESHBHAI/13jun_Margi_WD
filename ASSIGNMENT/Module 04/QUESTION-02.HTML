What is the use of isNaN function?

The isNaN function is used to determines whether a value is an illegal number or not a number.

This function returns true if the value equates to NaN . otherwise is return false.

Syntax : isNaN(value)

Example :

isNaN(10)__code
false __output

isNaN("it is function")__code
true __output
